## 常用
Ctrl + C
中斷目前畫面上操作 (中斷前景程序 process)

Ctrl + D
離開 (等同 exit 指令 + Enter)

Ctrl + Z
將目前畫面上操作切到背景 (不中斷操作)

Ctrl + L
清理畫面 (等同 clear 指令)

Ctrl + R
根據關鍵字查詢過往輸入的 Linux 指令

## 移動光標
Ctrl + A
移到行首

Ctrl + E
移到行尾

Ctrl + XX
在命令行尾和光標間切換

## 刪除字
Ctrl + W
刪除光標前的一個單字，這樣就不用每次都要按折 delete 一個一個字刪除了

Ctrl + K
刪除光標後所有字，搭配 Ctrl + A 可以刪除整行

Ctrl + U
刪除光標前所有字，搭配 Ctrl + E 可以刪除整行
